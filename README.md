Данный репозиторий представляет исходные коды пакета pysnap

PySnap — кроссплатформенный инструмент для быстрой настройки и управления виртуальными машинами для VirtualBox. PySnap использует вызовы [VBoxManage](https://www.virtualbox.org/manual/ch08.html) для выполнения настроек.

---

## Реализованные инструменты:

### vbsnap — инструмент для настройки ВМ

**Использование**:
 + vbsnap — выводит данное сообщение
 + vbsnap list — возвращает список настроенных ВМ с ключевыми параметрами:
   + Имя ВМ
   + Порт подключения к CLI машины после запуска
   + Сетевые подключения
 + vbsnap [<image>.ova | <image>.ovf] — Импортирует указанный образ в систему
 + vbsnap <VM> — Возвращает информацию о выбранной виртуальной машине:
   + Имя ВМ
   + Порт подключения к CLI машины после запуска
   + Сетевые подключения
 + vbsnap <BaseVM> <CloneVM>[:<Port>] [<eth1-net> [<eth2-net> [<eth3-net>]]] — создает связный клон от выбранной ВМ:
   + Клон создается на базе существующего снапшота базовой машины. В случае его отсутствия снапшот создается автоматически;  
   + При отсутствии указания порта по умолчанию устанавливает значение на один больше максимального из установленных на другие ВМ;
   + При отсутствии указания сетевых подключений по умолчанию не включает их;
   + vbsnap erase [--all | <CloneVM>] — удаляет ВМ (конкретную или все) и все связанные с ней (ними) снапшоты
    
 + vbsnap --cli — Активирует интерфейс командной строки:
   + команды аналогичны, но не требуют явного укзаания «vbsnap»
   + Для просмотра данного сообщения необходимо вызывать «help»
   + Используйте «quit» или Ctrl+D для выхода из CLI
